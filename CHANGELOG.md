# Changelog

## 1.1.10

* Renamed project to Easy Dashboard.
* Fix: avoid race condition when creating a new project (reloadDatabase during save).
* Sidebar tree view with drag & drop, sort mode, project icons.

## 1.0.3 - 2023-06-19

* Middle clicking a project will open it in a new window.
* Right clicking a project will do nothing now.

## 1.0.2 - 2023-01-20

* Easy Dashboard will only auto-expose itself if the window is 100%
  empty when it spawns.

* Fix how Microsoft returns lowercase drive letters from their
  folder picker.

## 1.0.1 - 2022-02-05

* Menu option to open project in new window.
* Setting to open project in new window by default.

## 1.0.0 - 2022-01-14

* Initial release
